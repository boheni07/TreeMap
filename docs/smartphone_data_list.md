# 스마트폰 수집 데이터 목록

스마트폰은 정밀한 센서와 통신 모듈을 통해 사용자의 생활 방식, 주변 환경, 기기 상태 등 방대한 양의 데이터를 수집합니다. 본 문서는 수집되는 데이터를 주요 범주별로 분류하여 정리합니다.

---

## 1. 모션 및 관성 데이터 (IMU)
기기의 움직임과 방향을 감지하는 핵심 센서군입니다.
- **가속도계 (Accelerometer)**: 기기의 이동 속도 변화, 기울기, 진동 등을 측정 (활용: 화면 자동 회전, 만보기, 낙상 감지)
- **자이로스코프 (Gyroscope)**: 기기의 회전 각속도와 방향 변화를 측정 (활용: VR/AR, 게임 조작, 카메라 흔들림 방지)
- **자기계 (Magnetometer)**: 지구 자기장을 감지하여 방위각 측정 (활용: 나침반, 지도 앱의 방향 표시)

## 2. 위치 및 연결성 데이터
사용자의 위치 정보와 통신 환경을 파악합니다.
- **GPS/GNSS**: 위성 신호를 이용한 정밀 위치(위도, 경도, 고도) 정보
- **Wi-Fi 정보**: 연결된 공유기의 SSID, 신호 강도, 주변 네트워크 목록
- **Bluetooth**: 연결된 주변 기기 정보 및 비콘(Beacon) 신호
- **셀룰러 네트워크**: 기지국 ID, 신호 세기, 이동통신사 정보

## 3. 환경 센서 데이터
기기 주변의 물리적 환경 조건을 측정합니다.
- **조도 센서 (Ambient Light)**: 주변 밝기를 측정 (활용: 화면 밝기 자동 조절)
- **근접 센서 (Proximity)**: 통화 중 얼굴 접근 등을 감지 (활용: 통화 시 화면 끄기)
- **기압계 (Barometer)**: 대기압을 측정하여 고도 변화 파악 (활용: 층수 이동 감지, 기상 예측)
- **온도/습도 센서**: 기기 내부 또는 주변의 온습도 (일부 고성능 기기 지원)

## 4. 미디어 및 오디오 데이터
멀티미디어 기능을 통해 수집되는 입력 정보입니다.
- **카메라**: 이미지, 동영상, 뎁스(Depth) 맵 (활용: 사진 촬영, 안면 인식, 사물 측정)
- **마이크**: 주변 소리, 음성 명령 (활용: 통화, 음성 비서, 소음 측정)

## 5. 시스템 및 사용 기록 데이터
기기의 운영 및 사용 패턴에 관한 기록입니다.
- **배터리 상태**: 잔량, 전압, 온도, 충전 주기
- **저장 공간**: 총 용량, 사용 중인 용량, 파일 형식별 점유율
- **앱 사용 로그**: 앱 실행 시간, 빈도, 체류 시간
- **알림 데이터**: 도착한 알림의 내용 및 빈도

## 6. 생체 정보 데이터
사용자 인증 및 건강 관리를 위한 데이터입니다.
- **생체 인식**: 지문 인식 정보, 안면 스캔 데이터 (대부분 보안 영역 내 별도 저장)
- **건강 데이터**: 걸음 수, 심박수(기기별 상이), 수면 패턴

---

> [!IMPORTANT]
> **Antigravity 프로젝트 관련 핵심 데이터**
> 현재 진행 중인 수목 측정 프로젝트에서는 위 목록 중 특히 **가속도계(기울기), 카메라(이미지 분석), GPS(위치 정보)** 데이터를 융합하여 나무의 위치와 크기(DBH, 수고)를 정밀하게 산출합니다.
